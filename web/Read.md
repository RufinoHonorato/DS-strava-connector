# Basic setup

## Connection to Firebase
```firebase login```
```firebase init```
Deploy with
```firebase deploy```

For more info:
https://firebase.google.com/docs/web/setup


## Installtion

```npm install```

* Configure EsLint
```./node_modules/.bin/eslint --init```
* Go to VSCode settings (CMD + ,) and turn on 'auto-fix on save'
